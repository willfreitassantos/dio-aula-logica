# Dio Aula Lógica - Calculadora de Rankeadas

Projeto simples da DIO que calcula o rank de um herói a partir de vitórias e derrotas.

Como usar:

1. Pelo terminal (args): `node index.js <vitórias> <derrotas>`
2. Pelo terminal (interativo): `node index.js` e seguir as instruções.

Exemplo:

```
node index.js 50 10
```
