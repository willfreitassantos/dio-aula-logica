Este projeto em JavaScript classifica o nível de um herói com base na sua experiência (XP). O programa cria variáveis para armazenar nome e XP, percorre uma lista de heróis usando laço de repetição e utiliza estruturas de decisão para determinar o nível correspondente (Ferro, Bronze, Prata, Ouro, Platina, Ascendente, Imortal ou Radiante). Ao final, exibe no console uma mensagem informando o nome do herói e seu respectivo nível.

*Objetivos de aprendizado:*
- Praticar variáveis e operadores (comparação e lógicos)
- Utilizar laços de repetição para processar múltiplos heróis
- Implementar estruturas de decisão para classificar valores em faixas
